# Crypto Atlas

Веб-приложение для мониторинга криптовалютного рынка в реальном времени. Позволяет просматривать топ-50 монет, искать, фильтровать, сортировать и изучать детальную информацию с графиком цен.

## Стек технологий

- **React 19** + **TypeScript**
- **Vite** — сборщик
- **React Router v7** — маршрутизация
- **TanStack Query v5** — серверный стейт и кэширование
- **Recharts** — графики
- **Zod** — валидация данных
- **SCSS Modules** — стили
- **CoinGecko API** — источник данных

## Архитектура

Проект построен по методологии **Feature-Sliced Design (FSD)**:

```
src/
├── app/          # Инициализация: провайдеры, роутер, глобальные стили
├── pages/        # Страницы: CoinsPage, NotFoundPage
├── widgets/      # Составные блоки: Header, CoinsList, CoinDetailsPanel, MarketLayout
├── features/     # Фичи: поиск, фильтрация, сортировка, переключатель валюты
├── entities/     # Бизнес-сущности: Coin (типы, API, маперы, чарт)
└── shared/       # Переиспользуемое: UI-компоненты, API-клиент, утилиты, типы
```

## Возможности

- Список топ-50 монет с ценой, капитализацией, объёмом и суточным изменением
- Иконки монет из CoinGecko
- Поиск по названию и тикеру с debounce
- Фильтрация: все / растущие / падающие
- Сортировка: по рыночной капитализации, цене, объёму, изменению за 24ч
- Детальная панель монеты: цена, капитализация, объём, ранг
- График цены с переключателем периода: 24ч / 7д / 30д
- Переключатель валюты: USD / EUR / RUB
- Состояние фильтров, поиска и выбранной монеты сохраняется в URL

## Запуск

### Требования

- Node.js 18+
- API-ключ CoinGecko Demo ([получить бесплатно](https://www.coingecko.com/en/api))

### Установка

```bash
npm install
```

### Переменные окружения

Создать файл `.env` в корне проекта:

```env
VITE_COINGECKO_API_BASE_URL=https://api.coingecko.com/api/v3
VITE_COINGECKO_API_KEY=your_api_key_here
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Сборка

```bash
npm run build
```

### Предпросмотр сборки

```bash
npm run preview
```
